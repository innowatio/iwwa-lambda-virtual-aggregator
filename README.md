# iwwa-lambda-virtual-aggregator

Aggregator for virtual-pods

## Env variables

- `AWS_SECRET_ACCESS_KEY`
- `AWS_ACCESS_KEY_ID`
- `AWS_DEFAULT_REGION`
- `LAMBDA_NAME`
- `LAMBDA_ROLE_ARN`
- `S3_BUCKET`
- `READINGS_API_ENDPOINT`

- `__FUNC_CONFIG__MONGODB_URL`
- `__FUNC_CONFIG__DEBUG`

## to run tests

`npm run test`
